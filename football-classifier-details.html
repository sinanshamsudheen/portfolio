Football Player Classifier

Football Player Classifier

Developed an end-to-end Football Player Classifier that identifies football players from images using machine learning. This project covers the entire pipeline, from data collection to deployment on AWS EC2.

Key Features:

  • Automated Image Collection
    • Used Bing Image Downloader to create a dataset of footballer images
    • Organized images into appropriate categories for training
  • Machine Learning Model
    • Trained an SVM classifier to identify players like Messi, Ronaldo, Neymar, etc.
    • Applied Support Vector Machine algorithm for accurate classification
    • Implemented rigorous training and testing procedures
  • Image Preprocessing
    • Applied Wavelet Transform techniques to enhance image features
    • Optimized preprocessing pipeline for better model performance
  • Web Application
    • Developed a user-friendly UI for image uploads and real-time classification
    • Created responsive frontend with HTML, CSS, and JavaScript
  • Backend API
    • Built a Flask API that processes images and returns predictions
    • Implemented efficient request handling and error management
  • Cloud Deployment
    • Successfully deployed the model and API on AWS EC2
    • Ensured scalability and accessibility for all users

Tech Stack: Python (OpenCV, Scikit-Learn, Flask), Machine Learning (Support Vector Machine), Frontend (HTML, CSS, JavaScript), Cloud Deployment (AWS EC2, Flask API, Model Serialization with Joblib)

Impact & Learnings:

  • Improved computer vision skills by implementing image classification
  • Gained experience in full-stack AI application development
  • Learned cloud deployment strategies using AWS EC2
  • Enhanced understanding of model deployment and API integration

This project showcases my expertise in AI, machine learning, web development, and cloud deployment. It demonstrates my ability to build end-to-end ML solutions that solve real-world problems.
